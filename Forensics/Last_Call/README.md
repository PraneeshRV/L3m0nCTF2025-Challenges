# Last Call

**Category:** Forensics

**Points:** 750

**Author:** R0z4r1y0

## Description

The emergency response team recovered a damaged phone from a crime scene.

The last recording on the device is a short call made under extreme distress.
You can hear panic, hurried movement, and what sounds like an alarm-like signal repeating again and again.

Investigators believe the caller was trying to transmit something, not speak it.

The call abruptly ends.

Your task is to analyze the audio and recover the message that was sent.
